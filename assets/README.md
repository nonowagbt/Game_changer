# Assets de l'application

## Icône de l'application

L'icône de l'application doit être basée sur l'icône "fitness" utilisée dans l'écran de login.

### Spécifications requises :

1. **icon.png** : 1024x1024 pixels, format PNG
   - Utilisé pour iOS et comme icône par défaut
   - Doit être un carré avec l'icône "fitness" centrée
   - Fond transparent ou couleur de fond de l'application

2. **adaptive-icon.png** : 1024x1024 pixels, format PNG (pour Android)
   - Même design que icon.png
   - Utilisé pour Android avec fond adaptatif

3. **splash.png** : 1242x2436 pixels (ou dimensions similaires), format PNG
   - Écran de démarrage de l'application
   - Peut contenir le logo "Game Changer" avec l'icône fitness

4. **favicon.png** : 48x48 pixels, format PNG (optionnel pour web)
   - Icône pour la version web

### Design recommandé :

- Utiliser l'icône "fitness" de Ionicons comme base
- Couleur principale : la couleur primary de l'application
- Style moderne et épuré
- Lisible à petite taille

### Outils recommandés pour créer l'icône :

- Figma
- Adobe Illustrator
- Canva
- Online icon generators (comme IconKitchen)

### Note :

L'icône actuellement affichée dans le login est l'icône "fitness" de Ionicons. 
Vous devez créer une version personnalisée de cette icône pour l'utiliser comme icône de l'application.
